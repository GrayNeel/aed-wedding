### Front-End Tech stack

- Bootstrap 5.3.3
- AOS 2.3.4
- Fontawsome 6.5.1
- Normalize 8.0.1
- Canvas Confetti 1.9.2
- Google Fonts
- Vanilla JS

- Node.js
- Express.js
- MySQL
